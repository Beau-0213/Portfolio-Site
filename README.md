# Personal Portfolio Website

This repository contains my personal portfolio website, created using HTML and CSS and hosted through GitHub Pages. The site was designed and coded by me to showcase my projects, technical skills, and web development work in a clean and organized format.

## About the Project

The purpose of this portfolio site is to provide a central place to present my work, demonstrate my understanding of front-end web development, and highlight projects that reflect my technical abilities and interests. Rather than using a website builder or template, I developed the site myself using foundational web technologies.

## Built With

- HTML5
- CSS
- Git
- GitHub
- GitHub Pages

## Features

- Multi-page website structure
- Custom HTML and CSS written from scratch
- Project showcase section
- Embedded media content
- Clean and responsive layout
- Accessible content elements such as captions and descriptions where applicable
