Forward Fitness Club

Forward Fitness Club is a multi-page website I developed as part of my intro to website technology course assignment to demonstrate my understanding of foundational web design principles. The project focuses on structuring content with semantic HTML, organizing navigation across multiple pages, and creating a clean, user-friendly layout. I included this project in my portfolio to showcase my ability to build a complete website using core HTML concepts. This site was built using resources provided to me through the Cengage textbook for the course.
