Graphics Card Replacement

Video demonstraton

This video demonstrates the physical process of installing a dedicated graphics card into a desktop computer, including accessing the case, identifying the correct PCIe slot, and securing the component. It also shows connecting the appropriate display cables. I included this project in my portfolio to highlight my ability to work directly with computer hardware and perform component upgrades safely and effectively.

Context: This project demonstrates troubleshooting a desktop display issue and implementing a hardware solution.

Figure 1: Installing and configuring a dedicated graphics card.

Skills & Tools:

  • Graphics card (GPU) installation
  • PCIe slot identification and usage
  • PC hardware handling and safety
  • Internal component installation
  • Display cable connection (HDMI / VGA)
  • Desktop computer assembly
  • Basic hardware configuration

What I Learned

This project strengthened my understanding of internal computer components and the proper procedures for installing hardware. It also reinforced the importance of handling components carefully and ensuring all connections are secure for proper system operation.
